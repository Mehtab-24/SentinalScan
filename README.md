# SentinelScan 🔒

SentinelScan is a DevSecOps web application that analyzes public GitHub repositories for security vulnerabilities using automated scanning tools.

## 🚀 Features
- Submit GitHub repository URLs
- Asynchronous security scanning
- Integration with Semgrep (SAST) and Trivy
- Severity-based scoring system
- Scan history tracking

## 🛠️ Tech Stack
- Backend: Spring Boot (Java 17)
- Database: PostgreSQL
- DevOps: Docker, GitHub Actions (planned)
- Security Tools: Semgrep, Trivy

## ⚙️ Status
✅ Backend complete  
⏳ Frontend in progress  

## 📌 Note
Scanning tools (Semgrep, Trivy) need to be installed locally for full functionality.
